Nesta documentação você encontrará tudo o que precisa para integrar com a API da DiHub. Desenvolvida por desenvolvedores para desenvolvedores, nossa plataforma foi projetada para ser intuitiva e fácil de usar.

## <Icon icon="Rocket" type="solid"/>  O que é a DiHub?

A DiHub é uma plataforma que simplifica processos complexos. Percebemos que as soluções existentes eram excessivamente complexas:
- Documentações extensas e confusas
- Processos de homologação longos e burocráticos
- Múltiplas formas de realizar a mesma operação

Nossa solução? Uma plataforma que transforma a complexidade em uma API simples e intuitiva.

## <Icon icon="laptop" type="solid"/>  API descomplicada

Nossa API foi construída com princípios fundamentais para facilitar a integração.

1. **Baseada em intenção**: Cada endpoint representa exatamente o que você lê
   ```
   GET /credits/balance    // Consulta saldo
   POST /diligence/emission // Emissão de diligência
   ```

2. **Consistente**: Nossa API retorna sempre um padrão previsível, facilitando a implementação.

Para facilitar ainda mais, oferecemos suporte a modo de desenvolvimento.
