Nesta documentação você encontra o essencial para autenticar sua aplicação, consultar créditos, emitir diligências e baixar os arquivos gerados pela DiHub.

## Como começar

<Steps>
  <Step title="Crie seu token de API">
    Gere um token no dashboard e envie o valor no header `X-Api-Token`.

    <Check>
      Você encontra o passo a passo completo em [Autenticação](/introduction/authentication).
    </Check>
  </Step>

  <Step title="Valide seu saldo">
    Consulte o endpoint de saldo antes de iniciar fluxos em lote ou operações sensíveis a crédito.

    <Check>
      A consulta de saldo não consome créditos.
    </Check>

    <Info>
      Se você não tiver créditos disponíveis, acesse o [painel financeiro no dashboard](https://app.dihub.com.br/perfil/financeiro) e solicite a quantidade desejada.
    </Info>
  </Step>

  <Step title="Emita sua primeira diligência">
    Envie o documento e as comarcas desejadas para iniciar a emissão e acompanhe o status até a conclusão.
  </Step>
</Steps>

## Ambientes disponíveis

### Produção

```bash
https://api.dihub.com.br
```

### Homologação

```bash
https://homolog.api.dihubapp.com
```

<Warning>
O ambiente de homologação não fica liberado automaticamente. Você precisa entrar em contato com a equipe de desenvolvimento da DiHub para solicitar o acesso. O fluxo de liberação diretamente pelo próprio usuário ainda está em desenvolvimento.
</Warning>

## O que você pode fazer com a API

- Consultar saldo de créditos
- Emitir diligências com múltiplas comarcas
- Acompanhar o processamento das solicitações
- Listar e baixar certidões, relatórios e arquivos consolidados em PDF
