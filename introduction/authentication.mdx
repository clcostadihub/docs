<Tip>
  **Chave de API**: Sua credencial de acesso à API da DiHub. Esta chave identifica sua conta e autoriza suas requisições. **IMPORTANTE**: Sem a chave de API, as requisições serão recusadas.
</Tip> 

## <Icon icon="lock" type="solid"/>  Gerenciando chaves de API

Gerencie suas chaves de API diretamente em nossa plataforma. Você pode:
- Listar todas as chaves ativas
- Criar novas chaves
- Revogar chaves existentes

<Card title="Erro de Autenticação" icon="warning" horizontal>
  A API retornará o código HTTP `401` quando:
  - A chave de API não for fornecida no header
  - A chave for inválida
  - A chave tiver sido revogada
</Card>

Quando o token é inválido ou não está presente, a API retorna a seguinte resposta:

```json
{
  "error": "Unauthorized",
  "message": "Missing or invalid api token"
}
```

## <Icon icon="key" type="solid"/>  Criando chaves de API

<Card title="Boas Práticas de Segurança" icon="light-emergency-on" horizontal>
  * Armazene suas chaves em variáveis de ambiente ou gerenciadores de segredos
  * Nunca compartilhe suas chaves de API
  * A DiHub nunca solicitará suas chaves
  * Revogue imediatamente qualquer chave comprometida
</Card>

## Como criar sua chave de API pela plataforma

Siga estes passos no [Dashboard DiHub](https://app.dihub.com.br) para criar sua chave de API:

<Steps>
  <Step title="Acesse o painel de API">
    No menu da sua conta (canto superior direito), clique em **API** para acessar a página de Chaves de API.

    <Frame>
      <img src="/images/entrar-no-painel-de-api.png" alt="Menu Minha conta com opção API" />
    </Frame>
  </Step>
  <Step title="Crie um novo token">
    Na página **Chaves de API**, clique no botão **+ Criar Novo Token** no canto superior direito.

    <Frame>
      <img src="/images/criar-novo-token.png" alt="Botão Criar Novo Token na página de Chaves de API" />
    </Frame>
  </Step>
  <Step title="Copie e salve seu token">
    Após a criação, o token será exibido na tabela. Use o ícone de copiar ao lado do token para copiá-lo e armazene-o em um local seguro — você pode copia-lo sempre que necessário.

    <Frame>
      <img src="/images/copiar-e-salvar-o-token.png" alt="Tabela de tokens com ícone para copiar" />
    </Frame>
  </Step>
</Steps>

<Card title="Desativando um token" icon="trash" horizontal>
  Para desativar um token que não precisa mais ser usado, clique no ícone de lixeira na coluna **Ações** da tabela de Chaves de API. Tokens desativados não podem ser reativados.

  <Frame>
    <img src="/images/desativar-o-token.png" alt="Ações para desativar token" />
  </Frame>
</Card>

Após criar sua chave, inclua-a em todas as requisições no header `X-Api-Token`:

```bash
curl 'https://api.dihub.com.br/credits/balance' \
  --header 'X-Api-Token: YOUR_API_TOKEN'
```

## Ambientes

Use uma das URLs abaixo conforme o ambiente liberado para a sua conta:

**Produção**

```bash
https://api.dihub.com.br
```

**Homologação**

```bash
https://homolog.api.dihubapp.com
```

<Warning>
O ambiente de homologação depende de liberação pela equipe de desenvolvimento da DiHub. A ativação feita diretamente pelo próprio usuário ainda está em desenvolvimento.
</Warning>

A validação padrão do token é de 90 dias.
