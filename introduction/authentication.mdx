<Tip>
  **Chave de API**: Sua credencial de acesso à API da DiHub. Esta chave identifica sua conta e autoriza suas requisições. **IMPORTANTE**: Sem a chave de API, as requisições serão recusadas.
</Tip> 

## <Icon icon="lock" type="solid"/>  Gerenciando chaves de API

Gerencie suas chaves de API diretamente em nossa plataforma. Você pode:
- Listar todas as chaves ativas
- Criar novas chaves
- Revogar chaves existentes

<Card title="Erro de Autenticação" icon="warning" horizontal>
  A API retornará o código HTTP `401` quando:
  - A chave de API não for fornecida no header
  - A chave for inválida
  - A chave tiver sido revogada
</Card>

## <Icon icon="key" type="solid"/>  Criando chaves de API

<Card title="Boas Práticas de Segurança" icon="light-emergency-on" horizontal>
  * Armazene suas chaves em variáveis de ambiente ou gerenciadores de segredos
  * Nunca compartilhe suas chaves de API
  * A DiHub nunca solicitará suas chaves
  * Revogue imediatamente qualquer chave comprometida
</Card>

## Como criar sua chave de API

Siga estes passos no dashboard da DiHub:

<Steps>
  <Step title="Acesse a Seção de Integração">
    <Card>
      <Card title="Clique no botão 'Criar Chave'">
        Inicie o processo de criação de uma nova chave de API
      </Card>
    </Card>
  </Step>
  <Step title="Adicione uma Descrição">
    <Card>
      <Card title="Identifique sua chave">
        Adicione uma descrição clara para identificar o propósito desta chave
      </Card>
    </Card>
  </Step>
  <Step title="Copie sua Chave">
    <Card>
      <Card title="Armazene sua chave com segurança">
        Copie a chave gerada e armazene-a em um local seguro
      </Card>
    </Card>
  </Step>
</Steps>

Após criar sua chave, você pode começar a integrar com nossa API. Lembre-se de incluir a chave em todas as requisições no header `X-Api-Token`:

```bash
curl --location 'https://api.dihub.com.br/v1/credits/balance' \
--header 'X-Api-Token: 01KE2M9KJMA3AHKB45WE72SJGX:MTFiMTIzODctNDkzNy00OTBiLTkxMDItZjU3Y'
```

A validação padrão do token é de 90 dias.
